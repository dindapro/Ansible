# Writing and Running Playbooks

## CHECK SYNTAX
```bash
[student@workstation playbook-basic]$ ansible-playbook --syntax-check site.yml
playbook: site.yml
```

## EXECUTE YML
```bash
[student@workstation playbook-basic]$ ansible-playbook site.yml
```
